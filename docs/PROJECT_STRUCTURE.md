# Climate Trend Analyzer Project 🌍

## Folder Structure Purpose

- **data/**: Contains raw and cleaned datasets (CSV format).
- **notebooks/**: Jupyter Notebooks for exploratory data analysis (EDA) and experimental modeling.
- **src/**: Source code for the web dashboard (React components and hooks).
- **models/**: Saved machine learning models (.pkl or .onnx).
- **outputs/**: Exported analysis charts and summary tables.
- **images/**: Project screenshots and visualization assets for README.
- **reports/**: Final PDF/Markdown research reports.
- **docs/**: Technical documentation and API references.
- **server.ts**: Node.js backend for serving data and AI analysis.
- **main.py**: Core Python script for data processing (Standalone version).
